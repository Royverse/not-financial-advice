Data Flow Sequence

sequenceDiagram
    participant C as Cron (9:30/12:00/16:00)
    participant S as Scanner
    participant AV as Alpha Vantage
    participant X as Xpoz
    participant AI as Gemini
    participant DB as Supabase

    C->>S: Wake Up
    S->>AV: GET Top Gainers
    AV-->>S: [AAPL, TSLA, NVDA...]
    
    loop Top 2 Candidates
        S->>AV: GET RVOL Data
        S->>AV: GET Float Data
        S->>X: GET Sentiment
        S->>AI: Analyze All Data
        AI-->>S: Buy/Hold/Sell + Score
    end
    
    S->>DB: Save Opportunities
