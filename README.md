welcome to **not financial advice.** 

we use AI to analyze market trends so you don't have to doomscroll twitter all day. 

## features

- **AI Vibe Checks:** Gemini analyzes diagrams and sentiment.
- **Sentiment Scans:** We check X (Twitter) for real-time market sentiment.
- **Modern UI:** Clean, dark mode, distraction-free.
- **Export to Excel:** Download your analysis for offline review.

## getting started

1. **Clone this repo:**
   ```bash
   git clone https://github.com/your-username/not-financial-advice.git
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Set your secrets:**
   Create a `.env.local` file:
   ```env
   GEMINI_API_KEY=your_key_here
   ALPHA_VANTAGE_API_KEY=your_key_here
   ```

4. **Run it:**
   ```bash
   npm run dev
   ```
